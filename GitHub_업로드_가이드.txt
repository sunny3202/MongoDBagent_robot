🚀 GitHub 업로드 가이드
========================

📋 사전 준비사항
----------------
1. Git 설치: https://git-scm.com/download/win
2. GitHub 계정 로그인
3. Personal Access Token 생성 (필요시)

🔄 자동 업로드 (권장)
-------------------
1. "GitHub_업로드.bat" 파일을 더블클릭
2. Git 설치 확인 후 자동으로 업로드 진행

📝 수동 업로드 방법
------------------
1. Git 설치 확인:
   git --version

2. Git 초기화:
   git init

3. 사용자 정보 설정:
   git config user.name "sunny3202"
   git config user.email "your-email@example.com"

4. 원격 저장소 연결:
   git remote add origin https://github.com/sunny3202/MongoDBagent_robot.git

5. 파일 추가:
   git add .

6. 첫 번째 커밋:
   git commit -m "Initial commit: 로봇 데이터 MongoDB 시뮬레이터 v2.0"

7. GitHub에 업로드:
   git branch -M main
   git push -u origin main

⚠️ 주의사항
----------
- .gitignore 파일이 생성되어 있어 불필요한 파일들은 제외됩니다
- 로그 파일(*.log)은 업로드되지 않습니다
- __pycache__ 폴더는 제외됩니다

🔗 업로드 후 확인
----------------
https://github.com/sunny3202/MongoDBagent_robot 에서 확인하세요.

📞 문제 해결
-----------
- Git 인증 오류: Personal Access Token 사용
- 권한 오류: GitHub 저장소 권한 확인
- 네트워크 오류: 인터넷 연결 확인
